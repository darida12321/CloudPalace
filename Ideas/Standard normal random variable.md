This is a special case of a [[Normal random variable|normal random variable]], where $\mu=0$ and $\sigma=1$. [[Normal random variable|Normal variables]] can be converted to a **standard normal** one by $Y=(X-\mu)/\sigma$. Since this is a [[Linear transformation|linear transformation]], we have:
$$E[Y]=\frac{E[X]-\mu}{\sigma}=0,\qquad\text{var}(Y)=\frac{\text{var}(X)}{\sigma^2}=1$$

After converting a variable to **standard normal form**, we can look up its values using a **standard normal table** (also called a **z-table**).
$$P(X\leq x)=P\left(\frac{X-\mu}{\sigma}\leq\frac{x-\mu}{\sigma}\right)=P\left(Y\leq\frac{x-\mu}{\sigma}\right)=\Phi\left(\frac{x-\mu}{\sigma}\right)$$
